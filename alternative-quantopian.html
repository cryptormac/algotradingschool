There are other alternatives out there for Quantopian. We will first go over the basics of Quantopian then explain another option.

What is Quantopian?

“Quantopian provides free education, data, and tools so anyone can pursue quantitative finance. Select members license their algorithms and share in the profits.” - directly from the website.

There are two sides on how Quantopian earns a profit.
  1. Algorithmic traders who develop code and test for free. All members compete against other members. Anybody can join without any fancy qualifications or background checks.
  2. Institutional investors are managed by the algorithms developed by the traders who built those winning algorithms.
Quantopian offers knowledge on data, tools for anybody to learn quantitative finance.

What services does Quantopian offer?

Quantopian offers Education, Data, Trade Executions, Community Support, Backtesting and Optimization of algorithms.

Education

They way Quantopian provides education is through self-help articles.

Data

There are plenty of free data for US Stocks that goes all the way back to 2002. They have both 1 minute and daily timeframes that you can try out. They have a rich data set, you can upload your algorithimic strategy but the problem with this is that this strategy is stored on their server.

Trade Execution

All trades has to be executed using the one-minute bars. They also use the last traded price as the closing price. It is not possible to check data against other data sources.

Community

They have a community of researchers who are specialized in quantitative finance and algorithim trading. They share their ideas, code, algorithms, backtesting and strategies. They also have an active forum where members ask questions regarding subjects they are struggling with. Anybody can go there to learn and get questions answered.

Python Code and Simplicity

You need to know how to code in Python and need to have take other courses in finance to properly grasp what is being taught. They provide coding as well as a testing platform. However, they do not have personal instructors. Great thing is, there are plenty of resources available out there can teach you how to code in Python. The two main features that the Code Editor has is:
  1. Code folding
  2. Auto-complete

Indicators

Quantopian do have all the usual indicators but there is always a catch. Some indicators do require more data to see the actual result.

Backtesting

Try out the backtesting and see how you can improve trades by using historical data. They have many good statistics and charts to look into. A row for each trade is provided such as date of entry, date of exit, price of exit and return of trade. There is an option of backtesting and live trading that contains data for trading on the stock exchanges which supports time series analysis.

Optimization of algorithms

Optimization is usually associated with the testing process and Quantopian currently does not support optimization on parameters. The methods are mainly based on stock trading.

Conclusion

You can use their tools for free, but this is only for those people who are good at coding and can learn on their own. Software platform that users can use to research quantitative financial factors in equity markets around the world. This is an excellent resort for simple research and implementation of trading strategies. Quantopian has Python Notebook for backtesting and implementing strategies. Some professional traders have switched from the current platform to Quantopian as an alternative.
